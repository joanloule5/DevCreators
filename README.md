##  Technologies Utilisées
- HTML5 sémantique
- CSS3 avec Flexbox/Grid
- JavaScript Vanilla
- Font Awesome pour les icônes
- Git & GitHub pour le versioning

##  Fonctionnalités
1. **Site Responsive** - Adapté mobile/tablette/desktop
2. **Navigation avancée** - Menu dropdown, mobile-friendly
3. **Pages personnelles** - Chaque membre a sa page spécialisée
4. **Animations** - Transitions et effets interactifs
5. **Génération dynamique** - Contenu généré via JavaScript

## Workflow Git Collaboratif
1. **Branches feature** - Chaque fonctionnalité dans sa branche
2. **Pull Requests** - Revue de code avant intégration
3. **Merges** - Fusion contrôlée dans la branche main
4. **Commits sémantiques** - Messages clairs et descriptifs
